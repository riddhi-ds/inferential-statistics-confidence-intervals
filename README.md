# Statistics Journey - Day 10

# 📊 Statistical Inference with Confidence Intervals

## 🚀 Overview
This project demonstrates **Inferential Statistics in Python** using Confidence Intervals to estimate population parameters from sample data.

Instead of relying only on averages, this project focuses on understanding **uncertainty in data** and how reliable our estimates are.

Dataset used: Student Performance Dataset from Kaggle.

---

## ❗ Problem Statement

In real-world data, we rarely have access to the entire population.

While analyzing student performance, I realized that:

> A single average score is not enough to understand true population behavior.

So the key question becomes:

👉 *What is the likely range where the true population mean lies?*

---

## 🎯 Objective

- Understand Point Estimation
- Build Confidence Intervals using Z and t procedures
- Analyze effect of sample size on uncertainty
- Study effect of confidence level on interval width
- Compare group performance using statistical inference

---

## 📂 Dataset
Dataset used from :contentReference[oaicite:0]{index=0}:
- Student Performance in Exams Dataset
- Features: Math, Reading, Writing scores, Gender, Lunch type, etc.

---

## 🧠 Concepts Covered

### 📌 1. Point Estimation
Estimating population mean using sample mean.

### 📌 2. Confidence Intervals
Constructed using:
- Z-distribution (when σ is known)
- t-distribution (when σ is unknown)

### 📌 3. Standard Error
Measures variability of sample mean.

### 📌 4. Margin of Error
Defines the uncertainty range around estimates.

---

## 📊 Key Analyses

### ✔ Confidence Interval for Scores
- Math, Reading, Writing

### ✔ Effect of Sample Size
- Larger sample → narrower confidence interval

### ✔ Effect of Confidence Level
- 90% < 95% < 99% (increasing width)

### ✔ Z vs t Comparison
- t-distribution handles real-world uncertainty better

### ✔ Group Analysis
- Male vs Female performance comparison using CI

---

## 📈 Visualizations

- Confidence Interval Error Bar Plot  
- Confidence Level vs Margin of Error  
- Z vs t Distribution Comparison  
- Group-wise Performance Comparison  

---

## 🔑 Key Insights

- Confidence intervals provide a **range-based estimate**, not a fixed value  
- Real-world data always contains uncertainty  
- Increasing confidence level increases interval width  
- Larger sample size improves estimation accuracy  
- t-distribution is more realistic when population σ is unknown

This project demonstrates how **Inferential Statistics bridges the gap between sample data and population understanding**.

Instead of making exact predictions, we quantify **uncertainty**, which is critical in real-world data science.
